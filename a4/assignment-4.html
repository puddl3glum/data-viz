Huffman Encoding

Note: The question says there are 8 intensity levels, but the chart lists only 7, meaning that only three bits are necessary. In fact, only three bits are necessary for 8 values as well if three 0 bits are included.

The Huffman codes for each intensity are:

  • 0: 000000
  • 1: 01
  • 2: 000001
  • 3: 0001
  • 4: 001
  • 5: 00001
  • 6: 1

Before compression, there are 10,000 pixels. In that image, there are 7 intensities and to represent those intensities, 3 bits are needed (000 - 110). For fixed-width encoding, the total size of the image with be 30,000 bits, which is 3750B, or about 3.66KB.

The total size of the image after compression is 1825B, or about 1.78KB

The achieved compression ratio is 48.7%.

Rainbow Map

Rainbow maps can be visually unreliable if the luminance values do not transition smoothly with the colors used. If there is a sudden jump in the luminance values, it can seem as if there is a major jump in whatever data range is being represented.
